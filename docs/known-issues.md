# JYM Log 알려진 문제

기준 버전: `v0.2.0-rc.1`

## 미검증 환경

### iPhone Safari 및 iOS 설치형 PWA

현재 iPhone 실기기 테스트가 완료되지 않았다.

사전 대응된 항목:

- safe-area
- 입력창 16px 이상
- 모바일 키보드
- 화면 높이 변화
- 홈 화면 PWA
- 좌측 가장자리 뒤로가기 제스처

정식 호환을 보장하려면 실제 iPhone 테스트가 필요하다.

## 설정 스위치

다음 설정 스위치는 현재 화면상의 상태만 변경한다.

- 기본 휴식 알림
- 화면 꺼짐 방지
- 오프라인 우선 저장

현재 동작:

- 터치·마우스·키보드로 켜고 끌 수 있음
- 접근성 상태가 함께 변경됨

현재 제한:

- 선택 상태가 앱 재실행 후 저장되지 않음
- 실제 진동·화면 켜짐 유지·저장 정책과 연결되지 않음

실제 기능 연결은 후속 버전에서 진행한다.

## 동시 기기 수정

PC와 모바일에서 같은 운동을 동시에 수정하면 충돌 선택창이 표시된다.

안전한 사용 방식:

- 실제 운동 입력은 한 기기에서 진행
- 다른 기기는 기록 조회 또는 루틴 확인 중심으로 사용

## iOS 업데이트 동작

서비스 워커 업데이트 안내는 구현돼 있지만 iOS 홈 화면 PWA에서의 실제 동작은 아직 검증되지 않았다.

## 분석 범위

현재 분석은 완료 운동 기록을 기반으로 한 기초 통계다.

지원 항목:

- 주간 운동 횟수
- 완료 세트 수
- 총 운동 볼륨
- 일부 운동 중량 변화

아직 포함되지 않은 항목:

- RPE·RIR
- 예상 1RM
- 부위별 볼륨
- 장기 증량 추이
- 체성분 변화
