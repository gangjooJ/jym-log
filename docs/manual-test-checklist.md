# JYM Log 수동 회귀 테스트 체크리스트

## 테스트 기본 정보

- 테스트 버전: v0.1.1
- 테스트 날짜: 2026-07-13
- 테스트 기기: Samsung Galaxy S25   
- 운영체제: Android 16.0 (Onu UI 8.5)
- 브라우저: Samsung Browser
- 접속 방식:
  - [O] 일반 브라우저
  - [O] 홈 화면 설치형 PWA
- 테스트 담당자: GangJ(Developer)

---

## 1. 앱 로딩

- [O] 첫 화면이 정상적으로 표시된다.
- [O] 상단 앱 이름이 `JYM Log`이다.
- [O] 상단 로고가 `JL`이다.
- [O] 오늘 날짜가 자동으로 표시된다.
- [O] 화면 디자인이 깨지지 않는다.
- [O] 하단 메뉴가 표시된다.
- [O] JavaScript 오류로 빈 화면이 나타나지 않는다.

---

## 2. 설정과 버전

- [ ] 설정 화면으로 이동할 수 있다.
- [ ] 앱 이름이 `JYM Log`로 표시된다.
- [ ] 현재 버전이 `v0.1.1`로 표시된다.
- [ ] 업데이트 날짜가 표시된다.
- [ ] 설정 토글 버튼을 누를 수 있다.
- [ ] PWA 설치 안내 버튼이 작동한다.

---

## 3. 운동 시작

- [ ] `운동 시작하기` 버튼이 작동한다.
- [ ] 첫 운동이 벤치프레스로 표시된다.
- [ ] 운동 진행 순서가 표시된다.
- [ ] 목표 중량과 반복 수가 표시된다.
- [ ] 직전 수행 기록이 표시된다.
- [ ] 세트 목록이 정상적으로 생성된다.

---

## 4. 세트 입력

- [ ] 중량 감소 버튼이 작동한다.
- [ ] 중량 증가 버튼이 작동한다.
- [ ] 반복 감소 버튼이 작동한다.
- [ ] 반복 증가 버튼이 작동한다.
- [ ] 중량을 직접 입력할 수 있다.
- [ ] 반복 수를 직접 입력할 수 있다.
- [ ] 중량이 0kg 미만으로 내려가지 않는다.
- [ ] 반복 수가 0회 미만으로 내려가지 않는다.

---

## 5. 세트 완료

- [ ] 완료 버튼을 누르면 세트가 완료 상태가 된다.
- [ ] 완료된 세트의 디자인이 변경된다.
- [ ] 완료 버튼을 다시 누르면 완료가 취소된다.
- [ ] 세트 완료 상태가 저장된다.
- [ ] 앱을 다시 열어도 완료 상태가 유지된다.

---

## 6. 휴식 타이머

- [ ] 세트 완료 후 휴식 타이머가 시작된다.
- [ ] 타이머가 1초씩 감소한다.
- [ ] 30초 추가 버튼이 작동한다.
- [ ] 종료 버튼이 작동한다.
- [ ] 시간이 끝나면 안내 문구가 표시된다.
- [ ] 운동 화면 이동 후에도 앱이 오류 없이 동작한다.

---

## 7. 운동 이동

- [ ] 다음 운동 버튼이 작동한다.
- [ ] 이전 운동 버튼이 작동한다.
- [ ] 첫 운동에서 이전 버튼이 비활성화된다.
- [ ] 마지막 운동에서 버튼 문구가 `처음으로`가 된다.
- [ ] 현재 운동 위치가 앱 재실행 후 유지된다.

---

## 8. 운동 완료

- [ ] 운동 완료 버튼이 작동한다.
- [ ] 피로도 입력 창이 표시된다.
- [ ] 피로도 1~5를 선택할 수 있다.
- [ ] 기록 저장 및 완료 버튼이 작동한다.
- [ ] 완료 화면으로 이동한다.
- [ ] 완료 세트 수가 정확하다.
- [ ] 총 운동 볼륨이 정확하다.
- [ ] 벤치프레스 성공 여부가 반영된다.
- [ ] 다음 운동 추천 이유가 표시된다.

---

## 9. 기록 유지와 초기화

- [ ] 브라우저를 닫았다 열어도 기록이 유지된다.
- [ ] 홈 화면 앱을 닫았다 열어도 기록이 유지된다.
- [ ] 초기화 버튼을 누르면 확인창이 표시된다.
- [ ] 초기화 취소가 가능하다.
- [ ] 초기화를 승인하면 운동 기록이 삭제된다.

---

## 10. PWA

- [ ] 브라우저에서 홈 화면에 추가할 수 있다.
- [ ] 설치 이름이 `JYM Log`이다.
- [ ] 홈 화면 아이콘이 표시된다.
- [ ] 설치 앱이 독립 화면으로 실행된다.
- [ ] 주소창 없이 실행된다.
- [ ] 새 버전 배포 후 재설치 없이 업데이트된다.
- [ ] 인터넷을 끈 상태에서도 기본 화면이 열린다.

---

## 11. Android 확인

- [ ] Chrome 화면에서 하단 메뉴가 시스템 영역과 겹치지 않는다.
- [ ] 숫자 입력 키보드가 정상적으로 표시된다.
- [ ] 화면 회전 후에도 사용할 수 있다.
- [ ] 뒤로가기 버튼이 예기치 않게 앱을 종료하지 않는다.

---

## 12. iPhone 확인

- [ ] Safari 하단 메뉴와 앱 버튼이 겹치지 않는다.
- [ ] 노치 또는 Dynamic Island 영역과 겹치지 않는다.
- [ ] 숫자 입력 시 화면이 과도하게 확대되지 않는다.
- [ ] 키보드가 입력창과 버튼을 가리지 않는다.
- [ ] 홈 화면 웹앱에서 정상 실행된다.
- [ ] 앱 전환 후 다시 돌아왔을 때 화면이 유지된다.
